## QUESTION:
Write a Java program to find the floor value of a number using Math.floor().

Input	Result
7.9
Floor value of 7.9 is 7.0

## AIM:
To write a Java program to find the floor value of a number using Math.floor().

## ALGORITHM :
1.	Start the program.

2.	Import the necessary package 'java.util'

3.	Read a decimal number from the user.

4.Use the Math.floor() function to compute its floor value.

5.Display the result to the user.

## SOURCE CODE:
```
import java.util.*;
public class Maths{
    public static void main(String[]args){
        Scanner sc = new Scanner(System.in);
        double a = sc.nextDouble();
        double b = Math.floor(a);
        System.out.print("Floor value of " +a+ " is "+b);
    }
}
```
